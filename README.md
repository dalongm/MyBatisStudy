# MybatisStudy

根据[《Spring MVC+MyBatis开发 从入门到项目实战》](https://item.jd.com/12308496.html)一书示例代码更新而来，并添加关键代码注释。

## Spring MVC与MyBatis项目实战

### 数据库设置

使用Mysql数据库，在src/main/resources/db.properties中配置数据库连接，账户及密码。
数据表及数据在sql/fruit_manage.sql中。

### 运行环境

+ Eclipse Oxygen
+ JDK 1.8
+ Tomcat 8.0
+ Mysql 5.7
+ Maven 3.5.3
